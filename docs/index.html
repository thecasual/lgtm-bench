lgtm‑bench

How often does AI write insecure code?

A benchmark that asks models ordinary, security‑neutral coding questions, then statically grades whether the code they return is exploitable. 5,666 trials, 10 models, 3 vulnerability classes, 4 languages. Every number regenerates offline from committed data.
